Reflection
The reflection that I have learned this past semester is to use and apply all of the managing enterprises topics into real life scenarios. One of that, is the use of SSH or safe Shell. Through SSH we were able to create an environment that contains one control node and three host nodes. Through these nodes, we were able to formulate and create meaningful activities such as ansible basics that was vital from the midterm to the finals period wherein we are automating the installation and configuration of each host node so that we would not manually configure and install applications on these nodes. The essence of a Systems Administrator is shown by these activities, I was also enlightened by our professor the one and only Dr. Jonathan Taylar. Thank you for your guidance this past semester.
